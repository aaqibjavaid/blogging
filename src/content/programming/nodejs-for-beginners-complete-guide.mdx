## What Is Node.js?

Node.js is a JavaScript runtime built on Google's V8 JavaScript engine.

It allows developers to run JavaScript outside the browser.

Before Node.js, JavaScript was primarily used for frontend development.

Node.js changed that by enabling JavaScript to run on servers.

This means developers can use JavaScript for:

* Frontend development
* Backend development
* APIs
* Real-time applications
* Command-line tools

## Why Is Node.js So Popular?

Node.js became popular because it solves several common development challenges.

### JavaScript Everywhere

Developers can use a single language across the entire application stack.

Example:

```text
Frontend → JavaScript
Backend → JavaScript
Database Queries → JavaScript
```

This reduces context switching and simplifies development.

### Fast Performance

Node.js uses the V8 engine, which compiles JavaScript into highly optimized machine code.

This makes Node.js significantly faster than many traditional server-side technologies.

### Large Ecosystem

Node.js has access to npm (Node Package Manager), the largest software package ecosystem in the world.

Developers can install thousands of open-source libraries with a single command.

## How Node.js Works

Unlike traditional server technologies that create a new thread for each request, Node.js uses an event-driven architecture.

This approach allows Node.js to handle many connections efficiently.

Simplified workflow:

```text
Request
↓
Event Loop
↓
Process Task
↓
Response
```

This architecture makes Node.js particularly effective for:

* APIs
* Real-time applications
* Chat systems
* Streaming platforms

## Installing Node.js

Download Node.js from the official website and verify installation:

```bash
node -v
```

Check npm:

```bash
npm -v
```

If both commands return version numbers, Node.js is installed successfully.

## Your First Node.js Program

Create a file:

```js
console.log("Hello, Node.js!");
```

Run it:

```bash
node app.js
```

Output:

```text
Hello, Node.js!
```

Congratulations! You've just executed your first Node.js application.

## Understanding npm

npm stands for Node Package Manager.

It helps developers install and manage dependencies.

Initialize a project:

```bash
npm init -y
```

Install a package:

```bash
npm install express
```

This creates:

* package.json
* node_modules
* package-lock.json

These files are essential for managing project dependencies.

## Building a Simple Server

Node.js includes built-in modules that make creating servers easy.

Example:

```js
const http = require("http");

const server = http.createServer((req, res) => {
  res.write("Hello World");
  res.end();
});

server.listen(3000);
```

Running this code creates a web server on port 3000.

## What Is Express.js?

Express is the most popular Node.js framework.

It simplifies:

* Routing
* APIs
* Middleware
* Request handling

Install Express:

```bash
npm install express
```

Example:

```js
const express = require("express");

const app = express();

app.get("/", (req, res) => {
  res.send("Welcome to Express");
});

app.listen(3000);
```

Most modern Node.js applications use Express or similar frameworks.
